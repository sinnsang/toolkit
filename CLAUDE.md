# Claude Code 작업 지침

**작업 시작 전 반드시 `PROJECT_CONTEXT.md`를 먼저 읽으세요.**

이 파일에는 다음이 포함됩니다:
- 프로젝트 목표 및 기술 스택
- 단계별 개발 로드맵 (1~8단계, 각각 검증 기준 포함)
- 안드로이드 SMS 앱 특화 주의사항 (무한 루프, Play Store 정책 등)
- 코딩 원칙 (단순함, 외과적 변경, 검증 가능한 목표)

## 작업 규칙 요약

1. **단계 단위로 진행**. 한 번에 한 단계만. 단계 사이에 사용자 확인.
2. **빌드 자동 검증**. 코드 작성 후 `./gradlew assembleDebug` 실행. 실패 시 자체 수정.
3. **추측 금지**. 모호하면 멈추고 질문.
4. **단순함 우선**. 요청 외 기능 추가 금지, 추측성 추상화 금지.
5. **현재 단계는 PROJECT_CONTEXT.md 하단에 명시됨**. 다음 지시 받기 전까지 자체 시작 금지.
